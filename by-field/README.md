# Study by Field: Where to Go for Your Subject

Most of this handbook is organised by country. This section is organised the other way round, by what you want to study. Each file below covers the universities best known for one field in **both Germany and Italy**, what they're known for, and the practical steps to apply.

Use this section to shortlist universities for your subject, then go to the country guides for the full admissions, visa, and cost detail.

## How to use this section

1. Open the file for your field and read both the Germany and Italy sections.
2. Note which universities run your program in a language you can study in (English Master's exist widely; many Bachelor's, and German/Italian medicine, do not).
3. Confirm every deadline, fee, and entrance-test detail on the official source linked in each file, because these change every year.
4. Move to the country deep-dives for the rest of the process: [Germany universities](../germany/universities.md) · [Italy universities](../italy/universities.md) · [Germany admissions](../germany/admissions.md) · [Italy admissions](../italy/admissions.md).

## A note on accuracy

Which universities are strong in a field is well documented and stable, so this section states it plainly. Application specifics — deadlines, fees, entrance-exam dates and cutoffs — change every year and vary by program and nationality. For those, each file hedges and links the official source. Always confirm the current detail before you act on it.
